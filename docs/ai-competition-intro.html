AI COMIC BUILDER
全栈漫剧 / 动画短片生成工作台

从剧本到成片：
端到端 AI 漫剧流水线

面向创作者的一站式工具：剧本解析、角色一致性、智能分镜、首尾帧与参考帧、多模型视频生成、FFmpeg 合成与字幕烧录。密钥后端存储，数据默认本地化部署。

项目版本参考 README（v0.2.x） · 文档生成日期：2026 年 5 月 · Apache License 2.0

背景与问题

传统漫剧与短片制作链路长、环节割裂：剧本结构化、角色设定统一、分镜语言专业、关键帧与视频模型接口各异。独立创作者与小团队难以在可控成本下完成「从文本到连贯影像」的闭环。

本项目定位

AI Comic Builder 将多阶段 AI 能力与本地媒体管线整合为可编排的 Web 应用：用户上传或编写剧本后，可按阶段触发或批量执行生成任务，并在分镜看板中跟踪进度、对比版本、精细编辑单镜提示词与参考资源。

核心能力（功能亮点）

剧本与分集

  • 支持 TXT / DOCX / PDF 导入与可视化解析流程
  • 项目级分集、按集角色关联与复用
  • AI 辅助剧本生成与结构化拆分

角色与一致性

  • AI 从剧本提取角色与视觉描述
  • 四视图参考图（正 / ¾ / 侧 / 背）
  • 分镜帧生成时智能解析场景并挂载角色参考图

分镜与编辑

  • 专业镜头列表：构图、灯光、运镜等指令
  • 分镜抽屉、角色内联面板、看板视图
  • 分镜版本管理与单镜 AI 优化

影像与成片

  • 首尾帧或场景参考帧模式
  • 视频提示词生成与多供应商视频模型
  • 片段拼接、字幕烧录、素材打包下载

技术栈摘要

层级技术选型
框架Next.js 16（App Router）
前端React 19、Tailwind CSS 4、Zustand、Base UI
国际化next-intl（中文 / English / 日本語 / 한국어）
数据SQLite + Drizzle ORM；异步任务队列
AIAI SDK / 多供应商图像与视频 API；可配置提示词注册表
视频fluent-ffmpeg 封装拼接与字幕

差异化与可演示性

  • 流水线产品化：不是单点「文生图」Demo，而是覆盖导入、角色、分镜、帧、视频、合成的完整工作流。
  • 一致性设计：四视图 + 分镜帧生成时的角色参考图解析，降低跨镜头形象漂移。
  • 工程与安全：API Key 后端存储；日志对 Prompt 降敏；默认本地媒体路径，适合大赛演示与私有化说明。
  • 可扩展：Provider 工厂与提示词槽位化，便于接入新模型与调优比赛场景。

演示素材：README 中附有 Bilibili 系统介绍与版本更新视频链接，可作为现场演示辅助材料。
